# TinyPixel

TinyPixel is a free browser-based image compression website that helps users reduce image file size while maintaining good quality. It is designed for government forms, job applications, scholarship forms, visa applications, college admissions, passport photos, and many other online upload requirements.

## Features

- Compress JPG, JPEG, PNG and WEBP images
- Client-side image compression
- No registration required
- Mobile friendly
- Fast processing
- Privacy-focused (images processed in browser)
- SEO optimized pages
- Google AdSense ready
- Responsive design

## Available Tools

- Compress Image to 20KB
- Compress Image to 50KB
- Compress Image to 100KB
- Compress Image for Job Application
- Compress Image for Scholarship Form
- Compress Image for College Admission
- Compress Image for Visa Application
- Compress Image for Online Form
- Passport Photo Tool
- JPG to PNG Converter
- PNG to JPG Converter
- WEBP Converter
- WhatsApp Image Compressor

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/tinypixel.git
```

Open the project folder.

Run with Live Server or simply open **index.html** in your browser.
